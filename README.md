# Covid-19_Data_Science_project
The project analyzes Covid-19 data in Finland. Regression modeling is done with XGBRegression model that tries to predict correctly survivability of the people in the data set. The first goal of this project is to analyze Covid-19 data in Finland and make conclusions about it. The second goal of this project is to build a model that can forecast accurately deaths in each country. The regression modeling is done with XGBoost Regression model, but this project probably needs more rigorous statistical methods, like analyzing bounds to be more informative.
